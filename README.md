# MedFinder Privacy Policy Site

This repository hosts the public privacy policy for the MedFinder Android app.

MedFinder is an Android application for looking up official medicine information, CIMA/AEMPS package leaflets, and AI-assisted informational summaries. This site exists only to provide a public, stable, non-login privacy policy URL for app store review and user transparency.

## Public URL

```text
https://medix-code.github.io/medfinder-privacy/privacy/
```

## Languages

- Spanish: `https://medix-code.github.io/medfinder-privacy/privacy/es/`
- Catalan: `https://medix-code.github.io/medfinder-privacy/privacy/ca/`
- English: `https://medix-code.github.io/medfinder-privacy/privacy/en/`

## Contents

The policy covers:

- App purpose and medical disclaimer.
- Camera usage for barcode scanning.
- Google sign-in and Google Play Billing.
- CIMA/AEMPS medicine data usage.
- AI-assisted query processing.
- Local storage, account data, retention and deletion.
- Security measures, user rights and minors.

## Security Notes

This repository must not contain secrets, tokens, keystores, private API keys, user data, production logs or internal configuration values.

## Deployment

The site is served with GitHub Pages from the `main` branch and repository root.
